# Archive

This folder keeps historical project material that is useful for context but is not the recommended entry point for running the cleaned prototype.

## Contents

- `pycharm-projects/`: tracked files from the old PyCharm workspaces, including intermediate scripts, setup experiments, and the original saved TensorFlow model.
- `legacy/`: the original final script and other one-off historical files kept out of the project root.
- `reports/`: the original PDF report and work journal.
- `presentation/`: the original slide deck.
- `reference-materials/`: loose historical media files used in the project.
- `project-agreement/`: the original agreement and concept material.
- `workspaces/`: an additional collection of old workspace snapshots that used to live at the repository root.

The original final script now lives here:

- `legacy/MA_WG_2024_Haag_Jan_Werk_17032005.py`
